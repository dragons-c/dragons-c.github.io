Spring

一、IOC（控制反转）

  控制反转是一种设计思想，就是将原本在程序中需要手动创建对象的控制权，交由Spring框架来处理。
  正控：若要使用这个对象，需要自己去负责创建。
  反控：若要使用这个对象，只需要从Spring容器中获取需要使用的对象，不关心对象的创建过程，也就是把创建对象的控制权反转给了Spring框架。
  代码：pom依赖: 

<dependency>  
  <groupId>org.springframework</groupId>  
  <artifactId>spring-webmvc</artifactId>  
  <version>5.2.0.RELEASE</version>  
</dependency>

  aplicationContext: 

<context:component-scan base-package="com.pojo"/>//包扫描

Linux学习笔记

一、硬件部分

(1)cpu
   cpu内部含有指令集，不同的指令集会导致cpu工作效率的高低。cpu性能的比较还有cpu的频率(频率就是cpu每秒钟可以进行的工作次数)，所以频率高表示这块cpu单位时间内可以做更多的事情。另外，不同cpu型号大多具有不同的针脚，能够搭配的主板芯片组也不相同，所以将主机进行升级时，不能只考虑cpu,还需留意主板支持的型号。
(2)内存
  内存需要注意带宽(频率×位宽)和频率的型号，还需要考虑容量。
(3)显卡
  显卡又称为VGA，对于图形影像的显示扮演着相当关键的角色。一般对于图形影像的显示重点在于分辨率和颜色深度，因为每个图像显示的颜色会占用内存，因此显卡上面会有集成内存并被称为显存，这个显存容量将会影响到你的屏幕分辨率和颜色深度。
(4)硬盘
  现在硬盘大多是SATA接口(SATA是Serial ATA的缩写，即串行ATA。它是一种电脑总线，主要功能是用作主板和大量存储设备（如硬盘及光盘驱动器）之间的数据传输)。
  固态硬盘(SSD)最大的好处是没有马达不需要转动。而是通过闪存直接读写的特性，因此除了没数据延迟且快速之外，还很省电。SSD的主流容量还是在128~256GB，而机械硬盘(HDD)容量已经达到2TB以上。一般固态硬盘(SSD)用作系统盘，机械硬盘（HDD）作为数据存储。
(5)网卡和主板
  需要注意网卡、显卡插在主板上的位置。
(6)主机电源
  要选择稳定性高的电源。

二、主机规划与磁盘分区

  正常的物理机器大概使用的都是/dev/sd[a-p]的磁盘文件名。至于虚拟机环境中，为了加速，可能就会使用/dev/vd[a-p]这种设备文件名。
  (1)MBR
   MBR磁盘分区表用来处理启动引导程序和分区表。而启动引导程序记录区和分区表通通放在磁盘的第一个扇区。但是无法使用2.2TB以上的磁盘容量，MBR仅有一个区块，被破坏后经常无法或很难恢复。
  (2)GPT磁盘分区表
   可以支持更大的磁盘容量。

一、标题

在#后面加个空格再写文字，一个#号是一级标题，两个是二级标题，以此类推，可以有六个标题。
比如：
   # 一级标题
   ## 二级标题
显示效果：

一级标题

二级标题

二、字体与换行

(1)换行
  行尾加两个空格就可以换行了
(2)加粗
  左右两边用两个*包起来就可以了
例如：
   **这是要加粗的字体**
显示效果：
  这是要加粗的字体
(3)空格
  &加emsp;等于两个&加nbsp;
(4)斜体
  需要倾斜的文字左右两边用一个*包起来
显示效果：
  这是要倾斜的文字
(5)斜体加粗
  要斜体加粗的文字左右需要使用三个*包起来
显示效果：
  这是要斜体加粗的文字

三、引用

在需要引用的文字前面加>就可以了，引用可以多层嵌套，如加两个>>或三个>>>
例如：
  >第一层引用内容
  >>第二层引用内容
显示效果：

第一层引用内容

第二层引用内容

四、分割线

三个或三个以上的*

五、超链接

[链接名]（超链接地址“超链接描述”）
例如：
  [曹某某博客](http://dragons-c.top/)
显示效果：
   曹某某博客

六、代码

(1)单行代码
  单行代码之间使用反引号包起来就可以了
例如：
  `System.out.println(“zzzz”);`
显示效果:
   System.out.println("zzzz");
(2)多行代码
  代码使用三个反引号包起来且两边的反引号都单独占一行
例如：
  ```
  while(true)
  {
     System.out.println(“Hello World”);
  }
  ```
显示效果： 

while(true)      
{  
    System.out.println("Hello World");  
}

七、列表

(1)无序列表
  *加空格
示例：
   列表内容1
   列表内容2
  * 列表内容3
(2)列表嵌套
  在上一级和下一级之间敲三个空格即可
示例：

  1. 有序列表1
      无序列表1
          无序列表2
  2. 有序列表2
      无序列表1
          无序列表2
