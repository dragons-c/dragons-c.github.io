MarkdownPad2使用

2018-08-14
本文总阅读量次

一、标题

在#后面加个空格再写文字，一个#号是一级标题，两个是二级标题，以此类推，可以有六个标题。
比如：
   # 一级标题
   ## 二级标题
显示效果：

一级标题

二级标题

二、字体与换行

(1)换行
  行尾加两个空格就可以换行了
(2)加粗
  左右两边用两个*包起来就可以了
例如：
   **这是要加粗的字体**
显示效果：
  这是要加粗的字体
(3)空格
  &加emsp;等于两个&加nbsp;
(4)斜体
  需要倾斜的文字左右两边用一个*包起来
显示效果：
  这是要倾斜的文字
(5)斜体加粗
  要斜体加粗的文字左右需要使用三个*包起来
显示效果：
  这是要斜体加粗的文字

三、引用

在需要引用的文字前面加>就可以了，引用可以多层嵌套，如加两个>>或三个>>>
例如：
  >第一层引用内容
  >>第二层引用内容
显示效果：

第一层引用内容

第二层引用内容

四、分割线

三个或三个以上的*

五、超链接

[链接名]（超链接地址“超链接描述”）
例如：
  [曹某某博客](http://dragons-c.top/)
显示效果：
   曹某某博客

六、代码

(1)单行代码
  单行代码之间使用反引号包起来就可以了
例如：
  `System.out.println(“zzzz”);`
显示效果:
   System.out.println("zzzz");
(2)多行代码
  代码使用三个反引号包起来且两边的反引号都单独占一行
例如：
  ```
  while(true)
  {
     System.out.println(“Hello World”);
  }
  ```
显示效果： 

while(true)      
{  
    System.out.println("Hello World");  
}

七、列表

(1)无序列表
  *加空格
示例：
   列表内容1
   列表内容2
  * 列表内容3
(2)列表嵌套
  在上一级和下一级之间敲三个空格即可
示例：

  1. 有序列表1
      无序列表1
          无序列表2
  2. 有序列表2
      无序列表1
          无序列表2
